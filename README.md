# CSSECDV
Case Study Group 7

Tech Stack:
MongoDB, Next.js, React (+ Tailwind CSS), Node.js

Instructions:
1. cd ticketing-system
2. npm install
3. Create a file named .env.local inside ticketing-system folder
4. Copy and paste the format of the .env.example file inside the .env.local file and replace the placeholder mongoDB URL with the actual mongoDB URL
5. npm run dev
